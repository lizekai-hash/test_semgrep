### C++ 贪吃蛇游戏

一个在终端运行的高级版贪吃蛇，使用 C++17 编写，可在 Windows 的 Git Bash / MinGW 环境下编译运行。

#### 功能特性
- **主菜单**：开始游戏、设置、退出  
- **设置选项**：
  - 地图大小：小 / 中 / 大
  - 初始速度：慢 / 中 / 快
  - 规则：撞墙是否死亡（穿墙模式 / 传统模式）
- **游戏内功能**：
  - 实时键盘控制（WASD 或方向键）
  - 正常食物（`*`）+ 稀有食物（`$`，额外得分且存在时间有限）
  - 随机障碍物（`X`）
  - 得分与等级系统（长度越长速度越快）
  - 暂停与继续（按 `P`）
  - 即时退出当前局（按 `Q` 回到菜单）
  - 最高分存档（本地 `highscore.txt`）

#### 依赖环境
- C++17 编译器（推荐 `g++`，如 MinGW-w64）
- 终端支持 ANSI 转义序列（Git Bash 默认支持）

#### 编译
在项目根目录执行：

```bash
make
```

成功后会生成可执行文件：

- Windows：`snake.exe`

#### 运行

```bash
./snake
```

#### 操作说明
- **方向**：`W` / `A` / `S` / `D` 或 方向键 ↑ ← ↓ →
- **暂停/继续**：`P`
- **本局直接结束并返回菜单**：`Q`
- **回车键**：在菜单中确认选项

#### 可能的扩展方向（可自行二次开发）
- 多人对战模式（双蛇）
- 道具系统（减速、穿墙、无敌等）
- 关卡编辑器
- 使用图形库（如 SDL2 / SFML）做成窗口版
